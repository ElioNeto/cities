# Cities - City Builder 2D no GameMaker Studio 2

Jogo de simulação e construção de cidades em 2D inspirado em Cities: Skylines, desenvolvido em GameMaker Studio 2.

## 📋 Sobre o Projeto

Cities é um simulador de construção e gestão de cidades com visão top-down (2D), utilizando grade quadrada de 128x128 células. O jogo foca em mecânicas simplificadas mas profundas o suficiente para oferecer uma experiência envolvente de gestão urbana.

## 🎮 Funcionalidades Principais

- **Sistema de mapa e grade**: Mapa 128x128 com tiles 2D
- **Construção de estradas**: Drag-and-drop com snap à grade, tipos variados
- **Zoneamento**: Residencial, comercial e industrial com crescimento automático
- **Infraestrutura**: Rede elétrica, água e serviços públicos
- **Gestão financeira**: Impostos, orçamento e empréstimos
- **População**: Simulação de cidadãos e satisfação
- **Interface**: Câmera com zoom, modos de visualização e feedback

## 🚀 Como Usar

1. Clone o repositório
2. Abra o projeto no GameMaker Studio 2
3. Execute o jogo
